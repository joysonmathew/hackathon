# Experiment No: 05
# Docker Image

## Aim
Create a Docker image for an application stored in a local repository.

---

# Docker Installation

1. Open Chrome and search:
   - Docker install for Windows
   - Docker Desktop for Windows – x86_64

2. In Command Prompt run:

```bash
wsl --update
```

---

# Python Install

1. Create a folder named `DevOps` on Desktop.
2. Open the folder in VS Code.

## sample.py

```python
print("Hello World")
```

---

# Dockerfile

Create a file named `Dockerfile`

```dockerfile
FROM python

WORKDIR /app

COPY . /app

CMD ["python", "sample.py"]
```

---

# Terminal Commands

## Check Versions

```bash
python --version
docker --version
```

## Run Python File

```bash
python sample.py
```

## Build Docker Image

```bash
docker build -t test1 .
```

## Run Docker Container

```bash
docker run --name cont1 test1
```

## Run Another Container

```bash
docker run --name cont test1
```

## Tag Docker Image

```bash
docker image tag test1 test1vp25/test1
```

## Push Image to Docker Hub

```bash
docker push test1vp25/test1
```

---

# Output

Docker image is successfully created and pushed to Docker Hub.
