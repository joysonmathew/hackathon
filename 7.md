# Jenkins Integration with Communication Channel and Email Notification

# Aim
To integrate a communication channel with Jenkins for project status updates and enable email notifications for build status.

---

# Prerequisites

Before starting, ensure the following are installed:

- Jenkins
- Java JDK
- Git
- Internet connection
- Gmail account (for email notification)

Official Jenkins Website:
https://www.jenkins.io/

---

# Step 1: Install Jenkins

## For Windows

1. Download Jenkins from:
   https://www.jenkins.io/download/

2. Install Jenkins.

3. Start Jenkins service.

4. Open browser:

bash
http://localhost:8080


---

# Step 2: Unlock Jenkins

1. Copy administrator password from:

bash
C:\Program Files\Jenkins\secrets\initialAdminPassword


2. Paste password into Jenkins setup page.

3. Install suggested plugins.

---

# Step 3: Create Jenkins Project

1. Open Jenkins Dashboard.
2. Click:
   text
   New Item
   
3. Enter project name.
4. Select:
   text
   Freestyle Project
   
5. Click:
   text
   OK
   

---

# Step 4: Configure Git Repository

Inside project configuration:

## Source Code Management

1. Select:
   text
   Git
   

2. Enter repository URL:

bash
https://github.com/username/project.git


3. Save configuration.

---

# Step 5: Build the Project

## Build Section

Add build step:

text
Execute Windows batch command


Example command:

bash
echo Build Started


Or for Maven project:

bash
mvn clean install


---

# Step 6: Install Email Extension Plugin

Go to:

text
Manage Jenkins
→ Plugins
→ Available Plugins


Search and install:

text
Email Extension Plugin


Restart Jenkins after installation.

---

# Step 7: Configure Email Notification

Go to:

text
Manage Jenkins
→ System


Find:

text
Extended E-mail Notification


---

# Step 8: Configure SMTP Settings

## Example Gmail SMTP Configuration

| Field | Value |
|---|---|
| SMTP Server | smtp.gmail.com |
| SMTP Port | 465 |
| Use SSL | Enabled |
| Username | yourmail@gmail.com |
| Password | Gmail App Password |

---

# Step 9: Generate Gmail App Password

1. Open Google Account:
   https://myaccount.google.com/

2. Enable:
   - 2-Step Verification

3. Go to:
   text
   Security → App Passwords
   

4. Generate App Password.

5. Use generated password in Jenkins SMTP configuration.

---

# Step 10: Test Email Notification

Click:

text
Test configuration by sending test email


Enter recipient email.

If successful:

text
Email sent successfully


---

# Step 11: Enable Email Notification for Project

Open Jenkins Project:

text
Project → Configure


Scroll to:

text
Post-build Actions


Select:

text
Editable Email Notification


---

# Step 12: Configure Email Recipients

Example:

bash
student@example.com


---

# Step 13: Configure Email Trigger

Common triggers:

- Success
- Failure
- Unstable Build

Select:

text
Failure - Any
Success


---

# Step 14: Save and Build Project

Click:

text
Build Now


---

# Step 15: Verify Email Notification

After build completion:

- Success email will be sent
- Failure email will be sent if build fails

Example Email:

text
Subject: Build Success - Jenkins Project

Project Build Completed Successfully.


---

# Step 16: Integrate Communication Channel

## Slack Integration Example

### Install Slack Notification Plugin

Go to:

text
Manage Jenkins
→ Plugins


Install:

text
Slack Notification Plugin


---

# Step 17: Create Slack Webhook

1. Open Slack Workspace.
2. Create Slack App.
3. Enable Incoming Webhooks.
4. Copy Webhook URL.

---

# Step 18: Configure Slack in Jenkins

Go to:

text
Manage Jenkins
→ System
→ Slack


Add:

| Field | Value |
|---|---|
| Workspace | Your Workspace |
| Credential | Webhook Token |
| Channel | #project-status |

---

# Step 19: Enable Slack Notification in Project

Open Project Configuration:

text
Post-build Actions
→ Slack Notifications


Enable:
- Notify Build Start
- Notify Success
- Notify Failure

---

# Step 20: Build Project and Verify Notifications

Click:

text
Build Now


Notifications will appear in Slack channel.
