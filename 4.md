# DevOps Lab

# Maven Project Creation with Required Dependencies for CI/CD Pipeline

## Aim

To create a Maven project with required dependencies and run a simple JSON-based Java application using Eclipse IDE.

---

# 1. Creating a Maven Project in Eclipse

## Steps

1. Open **Eclipse IDE**.
2. Click on **File → New → Maven Project**.
3. Check **Create a simple project (skip archetype selection)** if required and click **Next**.
4. In the **Filter** section type:

```
org.apache.maven.archetypes
```

5. Select:

```
maven-archetype-quickstart (1.4)
```

6. Click **Next**.

---

# 2. Provide Project Details

Enter the following details:

```
Group Id  : ex1
Artifact Id : ex1
Version : 1.0-SNAPSHOT
Package : ex1
```

Click **Finish**.

Eclipse will create the Maven project structure automatically.

---

# 3. Maven Project Structure

The project structure will look like:

```
ex1
 ├── src/main/java
 ├── src/test/java
 ├── pom.xml
```

---

# 4. Add Maven Dependency

1. Open a web browser.
2. Search for:

```
Maven Repository simple-json 1.1.1
```

3. Open the dependency section.
4. Copy the dependency code.

Dependency example:

```xml
<dependency>
    <groupId>com.googlecode.json-simple</groupId>
    <artifactId>json-simple</artifactId>
    <version>1.1.1</version>
</dependency>
```

5. In Eclipse open:

```
pom.xml
```

6. Paste the dependency inside the `<dependencies>` section.

Example:

```xml
<dependencies>

<dependency>
    <groupId>com.googlecode.json-simple</groupId>
    <artifactId>json-simple</artifactId>
    <version>1.1.1</version>
</dependency>

</dependencies>
```

Save the file.

---

# 5. Create JSON File

Create a new file:

```
emp.json
```

Add the following content:

```json
{
  "fname":"AI",
  "lname":"ML"
}
```

Save the file.

---

# 6. Create Java Test Class

1. Go to:

```
src/test/java
```

2. Right Click → **New → Class**

Enter:

```
Class Name : readex
```

Click **Finish**.

---

# 7. Java Program to Read JSON File

Add the following code inside **readex.java**.

```java
import java.io.FileReader;
import org.json.simple.JSONObject;
import org.json.simple.parser.JSONParser;

public class readex {

    public static void main(String[] args) {

        JSONParser parser = new JSONParser();

        try {

            Object obj = parser.parse(new FileReader("emp.json"));
            JSONObject jsonObject = (JSONObject) obj;

            String fname = (String) jsonObject.get("fname");
            String lname = (String) jsonObject.get("lname");

            System.out.println("First Name: " + fname);
            System.out.println("Last Name: " + lname);

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

Save the program.

---

# 8. Run the Program

1. Right click on **readex.java**
2. Select:

```
Run As → Java Application
```

---

# 9. Expected Output

```
First Name: AI
Last Name: ML
```

---

# Result

Successfully created a **Maven project in Eclipse**, added **JSON dependency**, created a **JSON file**, and executed the **Java program** to read JSON data.
