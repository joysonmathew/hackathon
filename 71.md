# Experiment 07

## Integrate communication channel with Jenkins for status of project and enable email notification for build

### Steps

1. Login to Slack

2. In Jenkins:
   - Manage Jenkins
   - Available Plugins → Slack Notification → Install

3. System Configuration:
   - Slack
   - Workspace name before `.slack.com`

4. Credentials:
   - Add → Global
   - Choose Secret Text

5. Open:
```text
https://my.slack.com/services/new/jenkins-ci
```

6. Get Secret → Add to Secret
   - ID: notification
   - Description: vvce

7. Default Channel Name:
```text
#devops-demo
```

8. Apply and Save
