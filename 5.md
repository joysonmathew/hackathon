# Docker Setup and Deployment Guide

## Objective
To create a Docker image for a simple test application stored in a local repository and run the application using Docker.

---

# Step 1: Install Docker

## For Windows

### Download Docker Desktop
Official Website:
https://www.docker.com/products/docker-desktop/

### Installation Steps

1. Download Docker Desktop.
2. Install Docker Desktop.
3. Enable:
   - WSL 2
   - Virtual Machine Platform
4. Restart the system.
5. Open Docker Desktop.

---

# Step 2: Verify Docker Installation

Open terminal and run:

bash
docker --version


Expected Output:

bash
Docker version 27.x.x


---

# Step 3: Create Project Folder

Create a new folder:

bash
mkdir myapp
cd myapp


---

# Step 4: Create Simple Test Program

Create a file named:

bash
test.py


Add the following code:

python
print("Docker Container Executed Successfully")


---

# Step 5: Create Dockerfile

Create a file named:

bash
Dockerfile


Add the following code:

dockerfile
# Base Image
FROM python:3.11

# Set Working Directory
WORKDIR /app

# Copy Files
COPY . .

# Run Python Program
CMD ["python", "test.py"]


---

# Step 6: Build Docker Image

Open terminal inside project folder and run:

bash
docker build -t testapp .


### Explanation

| Command | Purpose |
|---|---|
| docker build | Builds Docker image |
| -t testapp | Image name |
| . | Current directory |

---

# Step 7: Verify Docker Image

Run:

bash
docker images


Expected Output:

bash
REPOSITORY   TAG       IMAGE ID       CREATED       SIZE
testapp      latest    abc12345       1 minute ago  250MB


---

# Step 8: Run Docker Container

Run the container:

bash
docker run testapp


Expected Output:

bash
Docker Container Executed Successfully


---

# Step 9: Verify Running Containers

Run:

bash
docker ps -a


---

# Step 10: Remove Container

Find container ID using:

bash
docker ps -a


Remove container:

bash
docker rm <container_id>


Example:

bash
docker rm abcd1234


---

# Step 11: Remove Docker Image

Run:

bash
docker rmi testapp


---

# Important Docker Commands

| Command | Description |
|---|---|
| docker build -t image_name . | Build image |
| docker images | View images |
| docker run image_name | Run container |
| docker ps | Running containers |
| docker ps -a | All containers |
| docker rm container_id | Remove container |
| docker rmi image_name | Remove image |
